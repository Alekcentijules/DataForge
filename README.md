# DataForge

**CLI tool for automating data cleaning, validation, and reporting**

---

## Overview

DataForge helps freelancers, small businesses, and data enthusiasts transform messy CSV, log files, and tabular data into clean, structured JSON files and comprehensive reports.  
It reduces repetitive manual work, improves data accuracy, and saves hours of processing time.

---

## Features

- Validate data and catch errors automatically  
- Convert CSV and log files into structured JSON  
- Generate summary reports in CSV or HTML  
- Easy-to-use CLI interface  
- Scalable for larger datasets and future integrations
